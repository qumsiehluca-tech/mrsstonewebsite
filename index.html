A few things worth knowing

  • "Stone Cold Loans" is a placeholder. The name, logo (a plain square), phone, and email are easy to find-and-replace across all three.
  • The application form is a demo. All 4 steps validate and show a success screen, but nothing is sent anywhere yet — connecting it to a real inbox or CRM is a quick follow-up once a theme is chosen.
  • Theme 3's images are stand-ins. They're self-contained industrial graphics (no external files). Drop real photography into theme3-industrial/assets/ using the same filenames to swap them — or I can source photos once the theme is picked.
  • Compliance language is in. Each footer notes that Stone Cold Loans provides merchant cash advances, not loans — worth a lawyer's review before launch.
  • Everything is plain HTML/CSS/JS. No build step, no dependencies — opens by double-clicking index.html in any folder.
